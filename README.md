## Project Description
This project demonstrates basic concepts of web technologies including HTML forms,
tables, lists, and PHP backend scripting. It covers GET and POST methods and basic
client-server interaction.

## Technologies Used
- HTML
- PHP
- XAMPP
- Web Browser

## Features
- Registration form
- Login system
- HTML tables and lists
- PHP form handling

## How to Run
1. Install XAMPP
2. Place files in htdocs folder
3. Start Apache server
4. Open index.html in browser
